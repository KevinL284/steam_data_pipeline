Projetinho simples para treino e prática durante minha nova jornada de trabalho.

O que espero desse projeto?
Nada, só quero curtir o processo e constuir algo simples que seja aplicavel a outros contextos.

O que tem nesse projeto?
- Extração de dados da Steam usando a API pública.
- Transformação dos dados usando Pandas.
- Carregamento dos dados transformados em um banco de dados SQLite.
- Visualização dos dados usando Matplotlib e Seaborn.


O que eu não implementei nele ainda:
- Testes unitários.
- Documentação detalhada.
- Automatização do pipeline usando Airflow ou Prefect.
- Deploy do pipeline em um ambiente de produção.
- Monitoramento e alertas para falhas no pipeline.

O que eu quero implementar nele no futuro:
- Testes unitários para as funções de extração, transformação e carregamento.
- Documentação detalhada do código e do processo de ETL.
- Automatização do pipeline usando Airflow ou Prefect para agendar execuções regulares.
